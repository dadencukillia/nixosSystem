# NixOS Dots
My personal NixOS dots

`Font: JetBrainsMono Nerd, Noto`
`WM: Hyprland`
`Bar: AGS`
`Launcher: Rofi`
`Explorer: Thunar`
`Color scheme: Gruvbox Dark`

## Features
- Hyprland special keys for apps (like `SUPER+T` for Telegram, `SUPER+F` for Vesktop)
- `SUPER+A` for clipboard history
- Workspaces has their own icons on bar when applications are opened there
- Music display badge on bar that replaces window titles section when music plays
- Responsible bar buttons
- NvChad config with Nix Script LSP for root
